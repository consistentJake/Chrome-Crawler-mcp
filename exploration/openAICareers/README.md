## Usage

### 1. Scrape Jobs (`scrape_all_jobs.py`)
Scrapes all job postings from OpenAI careers page and saves HTML files.

```bash
python scrape_all_jobs.py
```
- Requires Chrome browser with the BrowserIntegration MCP
- Saves HTML files to `jobs/` folder
- Outputs `all_jobs_extracted.json` with extracted data

### 2. Reprocess HTMLs (`reprocess_jobs.py`)
Re-extracts job data from existing HTML files using updated parsing logic.

```bash
python reprocess_jobs.py
```
- Reads all HTML files from `jobs/` folder
- Regenerates `all_jobs_extracted.json`
- Use after updating `utils.py` extraction logic

### 3. Filter Jobs (`job_filter.ipynb`)
Interactive Jupyter notebook for filtering and exploring jobs.

```bash
jupyter notebook job_filter.ipynb
```
- Filter by: keywords, skills, location, team, compensation, years of experience
- Pre-computed search columns for fast filtering
- Displays results in paginated HTML table
